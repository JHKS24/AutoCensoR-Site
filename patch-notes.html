무엇이 바뀌었는지 짧게 남깁니다.

릴리스 자산과 소스 공개가 준비될 때마다 이 페이지에 사용자 기준의 변경점을 정리합니다.

  • 공개 배포 준비Windows 실행 파일을 CPU Only와 Full로 나누고, 모델 가중치는 별도 준비 방식으로 정리합니다.
  • 자동 저장 흐름 정리처리와 수동 보정 결과는 작업 흐름에 따라 저장되도록 설명을 명확히 했습니다.
  • 문서와 문의 흐름 분리홈페이지는 정적 파일로 무료 운영하고, 질문과 제안은 Tistory 댓글로 받는 구조로 정리합니다.

상세한 변경 내역은 GitHub Releases와 커밋 로그에도 남깁니다.
